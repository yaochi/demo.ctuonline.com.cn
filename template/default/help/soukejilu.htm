授课记录功能介绍

授课记录是专门为就讲师提供的一个功能，平台支持录入4种类型的授课记录，分别是授课申报 课程开发 课程研究 在线课程辅导，录入授课记录可以根据一定的规则活动培训是家园中的经验值。

如何创建授课记录

培训是家园的管理员可以为内部讲师和外部讲师创建授课记录，内部讲师可以自己创建授课记录 。

授课记录的经验值规则

1）、授课申报经验值 = 授课时长 x 单位课时经验值 x 培训班级别 x (评估结果/5) 

单位课时经验值：75 

培训班级别：集团级 2 

省级 地市级　1 

评估结果：1 - 5 

2）、课程开发经验值 = 单位课程开发积分 x 课程级别 x 参与度 

单位课程开发积分：500 

课程级别：集团级 2 

          省级、地市级 1 

参与度：参与人 80% 

牵头人 100% 

3）、课程研究经验值 = 单位课程研讨经验值 x 参与研讨次数 

单位课程研讨经验值：50 

4）、在线课程辅导经验值 = 单位在线课程辅导经验值 x 在线辅导次数 

单位在线课程辅导经验值:500
